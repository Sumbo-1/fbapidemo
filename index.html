facebook login

Don't waste precious hack time by writing
a login system from scratch.

We have libraries for nearly all platforms and languages, making it easy to use Facebook Login within your app.


We're using the Javascript SDK in this presentation for
the demos. Let's dive into an example!

Logged Out View

Show a logged in button since the user is not identified:

When the "logged in state" of the user changes (such as going from not logged in -> logged in), a JS function that you write is called. In this demo, that JS function fades out this box and fades in the "logged in" box below.

Logged In View

The user is logged in! You can access the user's unique user ID, name and profile pic. Persist the uid in your backend to identify a returning user.

Want to log out? Uninstall the app here.
